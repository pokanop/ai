# Modern Avatars

[← Back to Image Prompts](../README.md)

Clean, stylized 3D profile pictures with smooth geometric surfaces, vibrant gradient backgrounds, and a polished digital aesthetic. These avatars strike the balance between recognizable identity and artistic stylization — less photorealistic than a headshot, more personal than a generic icon. Perfect for accounts where you want to stand out without using an actual photograph.

**Best for:** Social media profiles · Gaming accounts · Slack/Discord avatars · App user profiles · YouTube channel art · Personal branding

![Avatar Example](../../../assets/placeholder-avatar.jpeg)

> **Sample prompt used to generate the above image (Nano Banana 2):**
> ```text
> Modern 3D avatar render of a red fox wearing a black turtleneck sweater and round tortoiseshell glasses, 1:1 square profile picture format. Stylized geometric features with clean smooth surfaces and a subtle rim light separating the character from the background. Vibrant glowing teal-to-purple gradient background. High-end octane render quality with soft subsurface scattering. Friendly, confident expression.
> ```

## 💡 Tips & Best Practices

- **1:1 is king**: Profile pictures are always square. Always specify `1:1 square` in your prompt.
- **Rim light is essential**: The phrase "rim light separating the character from the background" prevents the avatar from blending into the gradient.
- **Gradient backgrounds pop**: Two-color gradients (`coral-to-peach`, `teal-to-purple`, `electric blue-to-violet`) create a more dynamic, professional look than flat colors.
- **Consistency for teams**: When creating matching avatars for a team, specify "same art style, same rendering quality, same rim lighting angle" across all prompts. Use NB2's multi-turn: generate the first, then say "Create another avatar in exactly the same style for [next person]."
- **Common pitfalls**: Avoid mixing "realistic" and "stylized" — pick one direction. Saying "realistic 3D avatar" often produces an uncanny valley result.
- **Pairs well with:** [Cinematic Headshots](cinematic-headshots.md) (photorealistic version of the same subject), [Pixar / 3D Animation](pixar-3d-animation.md) (more expressive, animated style)
